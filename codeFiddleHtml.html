Browser Game

Along with Becky Gahart and Matt Cordier, Seth created a browser version of the board game Tsuro. The three of them created it with Node.JS and Sockets.IO. This was the first time any of three had worked with Node.js or Sockets.IO but they feel it was a successful project that they all enjoyed. While learning Node.js and Sockets.IO was a benefit, Seth feels the greatest benefit was the intrigate team work needed to make this program a success. Each member had to regularly update the other members to avoid using different nameing conventions or doing work twice. It was a project where each thing done directly related to the other members of the team.

Wordpress Site

Along with Becky Gahart, Scott Parker, and Matt Cordier, Seth created a fake company wordpress site. They developed the site as if they were a company of web developers looking for new clients. Seth focused on the javascript functionality of the contact page and the overall backend using PHP.

Code Fiddle

This project is an in browser coding playground. It allows programmers to write HTML, Javascript, and CSS into the browser and see the results immediatly without leaving the page. Seth created this project on his own using those same three languages as well as PHP and used a database to store user data.
